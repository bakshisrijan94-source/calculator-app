📟 Premium Calculator Web App










A modern, interactive calculator built with HTML, CSS, and JavaScript. Features a premium UI, calculation history, click-to-reuse results, and keyboard support.

🖥️ Features
Glassmorphism-inspired premium UI design
Calculation history panel
Click a history item to reuse results
Clear history button
Keyboard support for numbers, operators, Enter, and Backspace
Fully responsive and beginner-friendly

🛠️ Usage
Open index.html in your browser (or click the live demo).
Click buttons or use your keyboard to perform calculations.
Every calculation is saved in the history panel on the right.
Click a history item to reuse the result.
Use Clear History to reset all calculations.
