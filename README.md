📟 Premium Calculator Web App

A modern, interactive calculator built with HTML, CSS, and JavaScript. Features a premium UI, calculation history, click-to-reuse results, and keyboard support.
🖥️ Features
Modern, glassmorphism-inspired UI design
Calculation history to keep track of previous calculations
Click on a history item to reuse results
Clear history button to reset previous calculations
Keyboard support for numbers, operators, Enter, and Backspace
Fully responsive and beginner-friendly
⚡ Demo

Clone the repository and open index.html in your browser:

git clone https://github.com/bakshisrijan94/calculator-app.git
cd calculator-app
🛠️ Usage
Click the buttons or use your keyboard to perform calculations
Every calculation is saved in the history panel on the right
Click a history item to reuse the result
Use the Clear History button to reset all previous calculations
